# techAPJ's dotfiles

* __OS X:__ `bash -c "$(curl -LsS https://raw.github.com/techAPJ/dotfiles/master/bin/dotfiles.sh)"`
* __Ubuntu:__ `bash -c "$(wget -qO - https://raw.github.com/techAPJ/dotfiles/master/bin/dotfiles.sh)"`
